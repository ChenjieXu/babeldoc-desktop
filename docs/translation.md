# 翻译工作流

## 1. 添加文档

将 PDF 拖入文档队列，或点击 **选择 PDF 文件**。支持批量添加，单个文件上限为 500 MB。

## 2. 确认任务设置

主工作台中的选项只影响当前任务。开始前重点确认：

- 语言方向是否正确；
- 模型是否已选择；
- 页面范围是否符合预期；
- 是否需要双语、单语或两种输出；
- 是否使用页面交替排版；
- 是否需要自动术语抽取或 CSV 术语表。

## 3. 开始与取消

点击 **开始翻译** 后，任务状态区域会显示当前阶段和进度。长文档翻译期间可以请求取消；应用会等待工作线程安全退出，避免损坏输出文件。

## 4. 查看结果

翻译完成后，结果区域会列出生成的文件并提供打开输出位置的入口。

默认情况下，输出文件保存在原 PDF 旁边。若在高级设置中配置了输出目录，则以该目录为准。

## 输出类型

| 类型 | 说明 |
| --- | --- |
| 双语 PDF | 在同一份文档中保留原文与译文 |
| 单语 PDF | 主要呈现译文内容 |
| 页面交替 | 原文页与译文页按顺序交替 |
| 标准双语 | 使用 BabelDOC 的标准双语布局 |

!!! tip "异常文档"
    扫描件、复杂公式或特殊字体文档出现问题时，先在设置的 **文档处理** 页面逐项启用 OCR 或兼容选项，避免一次开启所有恢复开关。
