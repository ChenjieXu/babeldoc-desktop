# 常见问题

## 没有可选择的模型

打开 **设置 → 模型与服务**，为任一服务商添加模型并保存。主工作台会自动刷新模型列表。

## macOS 提示应用来自未知开发者

当前公开构建没有 Apple Developer ID 签名。请确认安装包来自本仓库的正式 Release，然后在系统设置的隐私与安全页面中确认打开。

## Windows 出现 SmartScreen

当前 Windows 构建没有 Authenticode 签名。请先核对下载来源和 `SHA256SUMS.txt`，再根据系统提示选择是否运行。

## 启动时出现多个终端或多个应用图标

请升级到最新版本。冻结应用会在导入 Qt 之前处理 Python 多进程子进程，子进程不应再注册为多个前台应用窗口。

## 输出路径出现只读文件系统错误

未配置输出目录时，新版本会把结果保存到原 PDF 所在目录，而不是文件系统根目录。若仍失败，请检查原文件目录是否具有写权限，或在 **高级设置** 中选择可写的输出目录。

## 查看日志

应用日志位于：

```text
~/.babeldoc-desktop/logs/
```

提交问题时，请删除日志中的文件路径、API 返回内容或其他敏感信息后再上传。

## 仍然无法解决

请在 [GitHub Issues](https://github.com/ChenjieXu/babeldoc-desktop/issues) 提交：

- 操作系统和架构；
- BabelDOC Desktop 版本；
- 可复现步骤；
- 已脱敏的错误日志；
- 不包含机密内容的截图。
